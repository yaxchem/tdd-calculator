Target Delivery Date (TDD) Calculator

Instructions:

For any Service request in JIRA, you can find the "Request submission date time" in the Details of the task. Enter that date into "Today's date". Then using the "Service Type" dropdown to choose the service type. The resulting target delivery date will be found in the "Target delivery Date" input. You can take that and copy it into the "Due Date" field in the JIRA ticket.

For Full Manual Evaluations:

This field is only used when you have a Full Manual Evaluation. Since a Full Manual Evaluation can be any range of pages, the date is updated dynamically. If this value is 10, the date for FME and Evaluations should be the same.
